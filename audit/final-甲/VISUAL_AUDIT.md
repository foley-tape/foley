## 逐条根因

### 视-1 整体发虚 —— 宪法缺口（半分辨率的锐度代价）
- **肉眼**（still-4/5）：即便"微距"帧，刻度/丝印/网格仍带一层柔化；纸面文字边缘不脆。
- **根因**：纸/图表 canvas **正确**按 dpr 设 backing store（`instruments.js:114/122/123` `dpr=devicePixelRatio; canvas.width=r.width*dpr`），VU 是 SVG 矢量——二者**本应锐利**。糊来自 `lens.js:57`：WebGL 镜头层 `canvas.width=Math.ceil(innerWidth/2)`（半分辨率）＋ `canvas.style.width=innerWidth+'px'`（拉回全尺寸），再经 `stage.css:401 #lens{mix-blend-mode:overlay}` 盖满整机。半分辨率的颗粒/暗角/抖动层在 2× 屏上被线性放大约 4×，overlay 混合把这层低频柔化印到满分辨率面板上 → **整机发虚**。叠 `#machine:34 filter:drop-shadow` 的整机滤镜缓冲，再软一档。
- **宪法**：镜头法明文要"3% 胶片颗粒（shader）＋暗角＋浅景深"——柔化**部分是设计**。但宪法**未设清晰度地板/半分辨率预算**。半分辨率是体温法（CPU<5%）的取舍。
- **裁**：宪法缺口。建议新修正案设"镜头层最低有效分辨率/仅在暗场与颗粒生效、不吃器件读数锐度"，把发虚与省电解耦。

### 视-2 阴影塌房 —— 违宪（近似）｜发布前必修候选
- **肉眼**（still-4/6）：VU、RECORDER 表头像**贴纸**压在面板上，几乎不在香槟铝上投下落地阴影；立体感塌陷。
- **根因**：`stage.css:185 .bezel` 的 box-shadow 只有 `0 1px 0 顶光边` + `inset 内阴影`——**没有把器件投到面板上的外 drop-shadow**。真实器件嵌在面板会在下缘/右缘留一道接触阴影。整机唯一的立体投影在 `#machine:34 filter:drop-shadow(0 46px 60px)`，那是整机对"桌面"的影，不是器件对面板的影。
- **宪法**：光法"面板从黑暗中浮出""全宇宙一种低角度暖光"——有明确光模型，器件本应各自落影。判**违宪（近似）**：光法在场，实现漏了器件级接触阴影层。
- **裁**：发布前必修候选。成本低（给 `.bezel` 加一层方向性外 drop-shadow，跟 `#keylight` 左上光源一致：右下更重）。这是船长"塌房"的直接解。

### 视-3 坐标纸违和 —— 宪法缺口
- **肉眼**（still-4/5/6）：网格规整均匀、偏"数字感"，不像真印刷坐标纸（墨色深浅不匀、纸纹透格）。
- **根因**：网格由 canvas 程序化等距细线绘（`instruments.js` 走纸增量绘制，纸面 dpr 正确故非糊），但网格是**纯几何**：等宽等色、无印刷抖动、无纸纤维透过格线的质感。
- **宪法**：材质法规定了铝/木/瓷/玻璃/墨，**未规定坐标纸的印刷材质**。判宪法缺口。
- **裁**：v1.x＋修正案（把"坐标纸=印刷介质，遵介质易朽/微不匀"写入材质法）。

### 视-4 纸张无光学反应 —— 违宪（近似）
- **肉眼**：纸面均匀受光，画外灯移动/天气变档时纸面无 sheen 位移。
- **根因**：`stage.css:216 .chart-bezel .glass` 与 `:208 .paper-shade` 是**静态烘焙渐变**（固定 112deg 斜光），`film.js:204-212` 胶印侧同样是写死的 `createLinearGradient`。它们**不订阅** `#keylight`/`#room[data-weather]` 的方向与强度——光变而纸面高光不动。
- **宪法**：光法"全宇宙一种光…洒上表面"是**动态光模型**（`#keylight` 会随 weather/idle 变）。纸面玻璃却是死渐变 → 违宪（近似）。
- **裁**：v1.x。让纸/玻璃高光跟 `--w-dim/--w-warm/--idle-dim` 联动。

### 视-5 VU 无玻璃罩反射 —— 违宪（部分兑现）｜发布前必修候选
- **肉眼**（still-4/5/6）：VU 是平奶油盘面＋深色圆角框，**无曲面玻璃罩的鼓面反光、无环境高光条**。像露天盘面，不像罩在烟色玻璃下。
- **根因**：玻璃**并非完全缺席**——`index.html:46` `<linearGradient id="vu-glass">` ＋ `:66 <rect fill="url(#vu-glass)">` 给了一层**平面斜向烟色渐变**；`film.js:116` 也照搬"玻璃两片"平面。但 linearGradient 只能造平板玻璃的匀色，**造不出曲面罩的 specular 高光条＋边缘折射＋微凸放大**。材质法要的是"玻璃**罩**"（dome，有曲率），实现只到"平面烟色 tint"。
- **宪法**：材质法**白纸黑字**"烟色玻璃**罩**表头"。判**违宪（部分兑现）**——tint 有，罩（曲面+反射）无。这是船长九条里最贴宪法明文的一条。
- **裁**：发布前必修候选。加一条曲面 specular（radial 高光条 + 顶缘亮线 + 底缘暗折射），VU 与 RECORDER 同治。

### 视-6 表针裸露无封闭感 —— 违宪（同 5 根因）
- **肉眼**：VU 针像浮在盘上，没有"隔着一层玻璃看针"的封闭/纵深。
- **根因**：合成序其实**对**——`film.js:116/121/128` 明确"底→针→玻璃两片"，针在玻璃层下。但玻璃层是弱 opacity 的平面渐变（视-5），**没有反射把针"压"进玻璃下**（真玻璃罩会在针上方投一道高光/微反光，制造"针在罩内"的线索）。故封闭感缺失=视-5 的直接后果。
- **裁**：随视-5 一并修（specular 到位，封闭感自来）。

### 视-7 ASK 开关扁平无厚度 —— 宪法缺口（熄灭态形体未规定）
- **肉眼**（three stills，ASK 均未点亮）：ASK=右侧深色竖药丸，熄灭时近乎纯平深色块，无圆柱厚度。
- **根因**：ASK 即 `#amber-tube`（`stage.css:312` width30×height128 圆角15=竖药丸）。**点亮态**有 filament/hotcore/外溢光（`:325-339`）造型充分；**熄灭态**只剩 `background:linear-gradient(105deg, rgba(48,34,20),rgba(24,16,9))`（近平面）＋`::before` 一道立缝高光——**无圆柱形体明暗**（玻璃管熄灭时仍该有左亮右暗的柱面反光＋端盖厚度）。船长看到的多是熄灭态（ASK 只在等你时才亮）。
- **宪法**：材质法/灯位法规定了点亮语义，**未规定灯的熄灭态该有多少"物件性"**。判宪法缺口。
- **裁**：v1.x＋修正案（"仪器熄灭态仍是物件，须保形体明暗"）。给 `#amber-tube` 熄灭底加柱面 radial + 端盖高光。

### 视-8 DONE·MAIN 刻字排版松散 —— 违宪（近似）
- **肉眼**（still-4/5）：DONE、MAIN 字距偏松、且像**浮在**铝面上而非**刻进**铝里。
- **根因**：两处。①`stage.css:176 .silk letter-spacing:0.30em`（`.small`:0.24em）——tracking 偏大读"松散"。②`text-shadow:0 1px 0 rgba(255,250,235,0.28)`=字下方一道**亮**边=**浮雕（emboss，凸起）**观感；而字法要"**如蚀刻在铝上**"（etch，凹进）——蚀刻在左上光下应是**上缘暗、下缘微亮**的内凹，现在的纯亮下边偏凸。
- **宪法**：字法明文"小号大写字距，如蚀刻在铝上；数字永不见于面板明处"。tracking 是宪法要的（"字距"），但"蚀刻"被做成了浮雕 → 违宪（近似）。
- **裁**：v1.x。收一档 tracking（0.30→0.22em 级）＋把 text-shadow 改内凹配方（上暗下微亮）。

### 视-9 DUB 键像廉价注塑无阻尼感 —— 违宪
- **肉眼**：DUB 键按下=瞬间下沉，回弹生硬，像廉价注塑键；无金属键的阻尼回弹。
- **根因**：`stage.css:255-274 #dub-key`。①按压 `:270 #dub-key:active{transform:translateY(2px)}` 走 `:268 transition:transform 0.1s ease`——**补间**，非弹簧-阻尼；无 overshoot/settle。②材质：顶 `inset 0 1px 0 rgba(255,250,235,0.4)` 亮高光＋圆角＋匀色渐变，读"塑料光泽"（虽有 `:258` 拉丝叠层，仍偏亮塑）。
- **宪法**：动法**明文**"一切服从惯性。指针走弹簧-阻尼（**禁补间、禁 linear**）；指示灯钨丝式衰减"。README:35 也承诺"the renderer adds no easing"。DUB 键作为物理操作件，按压用 `0.1s ease` 补间**直接违动法**。判**违宪**。
- **裁**：发布前必修候选（低成本）。按压走一小段弹簧-阻尼（下沉快、回弹带一次微 overshoot）；顶高光压暗、加接触阴影，脱塑料感。
