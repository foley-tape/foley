## 1. 总纲：一张床，几件事

v0 的病（船长亲耳诊断）：声音只有离散事件，曲线只有连续张力，两轴正交、互不解释。v1 的方子：**双层架构**。

- **连续层（床）**＝状态的身体。永远在响，编码 T / A / wow / phase / weather。它是"曲线的声音"。
- **离散层（前景）**＝事件的标点。拨弦、咔哒、和弦、跳针。它是"时刻的声音"。
- 三层信号旧教义映射：背景态＋天气态 → 床；事件与呼唤 → 前景。呼唤词汇 ≤4 铁律不变。
- 诚实条款延伸到响度：**床的能量包络必须诚实追随 T**——F5 的直接回应（执法：金测试㉛设计律；渲染侧 G4 informational 候裁）。

## 2. 连续层（床）规格（v1.1 重订）

### 2.1 结构：三层床（SOUND-R2；游戏音频正统手艺之上加"真实织体为体"）
| 层 | 内容 | 门控/驱动 | 执法 |
|---|---|---|---|
| **L1 织体体** | **真采样氛围织体为体**（room tone＋空气感＋磨损 crackle；CC0 vendored，`sound/assets/LICENSES.md` 逐条溯源；无资产时合成织体同构退路）；互不通约环长防察觉；repo-key 变奏（±1% 率＋环相位）——主力项目永远"听起来像回家" | 永远在（IDLE 唯余此层最弱态）；crackle ∝ T | 金测试㊴定标锁；G6 织体占用度 |
| **L2 和声垫** | 有音高内容，**三关铁律**：①≥3 声部失谐堆叠（±3–10 音分）②动低通（≥2 互质慢速自由 LFO）③轻饱和。**裸振荡器上总线在图内无路径**。电平永低于 L1（参数层执法） | WORKING 常驻，IDLE/DONE 无 | 金测试㊺三关自述；resolve 铁律检查 |
| S2 律动 | 慵懒 boom-bap 骨架（拍点稀疏，力度轻） | A > 0.3 淡入；密度 ∝ A | ㊴（crest 记账） |
| S3 张力弦 | 暗色弦，和声悬挂音比例 ∝ T；**同过三关** | T > 0.35 渐入 | ㊴㊺ |
| S4 保真总线 | 磁带老化：wow&flutter、主低通、高频搁架挂**音乐路**；**hiss/crackle 属介质噪声，出低通直达输出**（v1.1 修正：读头后噪声不该被"磁带变闷"再滤一次——R1 实证 18dB 设计摆曾被对消至 6.7dB） | T 与 wow 驱动 | ㊽ hiss 摆幅实证 |

### 2.2 映射表（起手值全部进 sound-params.json，可调；G3 守"渲染=设计记账"）
- **T → 和声张力**：sus/add 音概率 = T；**T → 滤波**：主滤波截止 8kHz→1.8kHz 线性下压（音乐路）；**T → 磁带磨损**：hiss/crackle 电平随 T 上行（具体 dB 域见 sound-params——白皮书不再复写数字，防文纸与参数漂移）。执法：G3 ±3dB、㊽。
- **A → 律动密度**：触发概率 0.2→0.9；tempo 恒定 **72 BPM**（执法：㉝）。
- **wow → 走带不稳**：pitch LFO 深度 3→22 音分（执法：㊴ 路径级；深度有界=注册表调制口）。
- **phase**：IDLE = 仅 L1 最弱态；WAITING = 床转半终止悬停；DONE = 正格终止 → **真静默 ≥4s**（执法：㉚ 设计层；渲染层 G1 覆盖停止路径——DONE 渲染静默**未执法**，候补门）。
- **weather → 场景**：档位切换只在小节边界执行；下行切换即"消散的表达"。（执法：**未执法**——场景档听感差异无机器判据，候体验门扩展）
- **调性**：每仓库一调（repoKey→五声音阶主音）（执法：㉞）。
- **平滑纪律**：一切连续参数过 slew（250–800ms），禁跳变；参数更新对齐 1/8 拍网格。（执法：结构性——applyBed 唯一写者+setTargetAtTime；无独立门）

## 3. 离散层（前景）规格

### 3.1 词汇表（≤6 乐音 ＋ 呼唤 3，超编即删）
| 事件 | 声音 | 备注 |
|---|---|---|
| WRITE-OK | 拨弦（Karplus 类） | targetHash 选音——文件的主题曲；力度/亮度 ∝ 当刻 T（F1） |
| WRITE/READ-FAIL | 低音区闷拨弦 | 音区分裂承载信息（盲听证明） |
| READ-OK | 纸页翻动（极轻，最先习惯化沉床） | foley 物理触感 |
| RUN-OK | 打字机回车铃（轻） | test 触发 RESOLVE 时让位给和弦 |
| SAVE-OK | 卡座按键咔哒＋低音锚 | S≥saveResolveMinS 才附和弦 |
| SPAWN | 新声部淡入一小节 | 多轨预演 |
| **RESOLVE** | 和弦解决（正格进行） | 呼唤级，直通不量化 |
| **STUCK** | 跳针（同槽卡碟专属） | 呼唤级，直通；同 sig 一响，CLEARED 前不复鸣 |
| **ASK** | 半终止动机（温柔穿透，频谱专区 2–4kHz） | 呼唤级，直通；礼貌性重复（90s 一次，音量不升级） |

量化：乐音级量化 1/8 网格宁迟勿早（执法：㉝）；呼唤级永不量化。

### 3.2 告警经济学（F4 机械化：习惯化曲线）
- 同类前景音滚动 60s 窗第 n 次：×0.85^(n−1)，下限＝沉床比（执法：㉜）。
- 新颖即重置；呼唤级与 DONE 永不习惯化。
- **已知张力（EAR-5 实证，候裁）**：60s 窗按听者时间，倍速回放下事件密度成倍、全类速沉地板——原速法（施工令 R2 §1）已把默认听感调回 1×；窗口语义修订仍开放。

## 4. 呼唤态纪律（重申）
词汇 ≤4；音量不是报警维度——靠音色新颖、前置微静默、频谱专区穿透；永不习惯化；永不量化。
**已知张力（R2 首轮实证，候裁）**：织体床把 1.2–2.2k 专区填实后，G5 带能量差判据失效（Δ−4.4dB）——呼唤的"穿透"须重新定义（瞬态显著度/更深 duck/专区让频），候架构师修订判据。
