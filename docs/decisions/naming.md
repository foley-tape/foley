# 命名决议 —— Foley（M2.0 §0.4–0.5 收录）

## 决议
- 项目名 **Foley**；首台机器（本仓的这台琥珀走带机）沿用血统名 **TAPE·ZERO** 作为"第一台 deck"的机型名。
- **域名**：注册 `foleytape.com`（与 org 同名、复合词即可保护标识）；`foley.fm` 入观察清单，发布挣到之后再谈。
- **迁移**：远端迁至 `foley-tape/foley`；**历史文档名（TAPE0_*）保留不改，零翻新税**。
- **名字敏感面冻结**（M1.9 §3 起）：不再新增烧录 TAPE·ZERO 字样的资产；面板丝印保留为占位；性格照构图避开特写机名。

## 名字的道理（README《Why "Foley"》定稿口径）
1930 年代，Jack Foley 看着画面、用手边的实物**现场同步**演出画面的声音——脚步、衣响、天气。这台机器为不可见的劳动做同一件事：看着你 agent 的会话，用磁带演出它的声音。

## 留痕
- 命名轮由架构师携联网调研主持（M1.9 §3 预告）；决议以 M2.0 §0.4–0.5 落笔。
- 命名轮全文纪要未见于船长桌面手令堆——如另有会议纪要，请架构师补投本目录。
