# Foley 唱片血统档案 v1
调研与整编：品味权威分支 · 2026-07-06
用途：出厂唱片终裁的证据基础 + 淘碟指南（RECORDS.md）底稿 + 委托案提案。血统终裁权在主理人。

## 一、铁律：可 vendor ≠ 免费
开源仓库里的音频文件本体 = 再分发。据此一票裁定：
- **出局**：Pixabay / Mixkit / YouTube 音频库 / Chillhop 等"免版税"目录——许可仅覆盖"用于作品中"，禁止音频文件再分发；一切 NC（非商业）许可同样出局（堵死变现路径）。
- **入闸**：CC0 / 公有领域（首选，零义务）；CC-BY（可再分发，须署名，备用）。
- **每首出厂曲目持护照上岗**：见第八节 PROVENANCE 模板。

## 二、选片法反转：买 hi-fi，卖 lo-fi
把 hi-fi 变 lo-fi 是机器的工作，不是采购标准。听样三判据（按序淘汰）：
1. **出厂磨损度**——自带 crackle/wow 越少越好；磨损是我们的墨水，别买来就磨旧的带子。
2. **旋律轮廓**——有可哼的动机，跳针啃上去 3 秒内可读。
3. **密度与循环性**——能长时间垫在工作下面；查头尾是否有干净的换面切点。

## 五、封井矿：FreePD（不入厂机）
官网已挂告别信正式关站（"运营 17 年后下线"），维基共享资源标注 2025 年起停运；GitHub 存有完整镜像（github.com/0lhi/FreePD，含 CC0 法律文本）。CC0 不可撤销，法律上仍可用，但原始许可页取证链已断——裁定：归"历史矿井"柜，出厂唱片不采；如采镜像曲目，须以 archive.org 历史快照补全护照。目录风格（Epic/Comedy/Scoring 为主）本也不合我们的宇宙。

## 七、委托案：「Foley 首版压盘」（强烈推荐执行）
- 对象：Holizna（首选，风格对口）；备选 Rrrrrose Azerty（FMA 站内联系）。
- 形态：赞助式委托，非 work-for-hire——付费请艺术家为 Foley 创作/挑选 1–2 首，**以 CC0 公开发布**（保住开源纯度），署名「Pressed for Foley by Holizna」，机身铭牌 + README + 关于页鸣谢。
- 预算级：$50–150。
- 同信索取书面确认三句：原创作曲、无未清理采样、无 AI 生成——溯源链的最后一环由艺术家亲笔闭合。
- 叙事收益：把"AI 工具"的焦虑叙事反转为"AI 工具给人类音乐家创造了一单活"，hero 物料与 HN 帖的现成弹药。
