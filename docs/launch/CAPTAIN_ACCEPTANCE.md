# FOLEY · 船长验收协议(唯一由人类执行的验收)

> 你是这个项目最后一个、也是最权威的传感器。截图骗过了所有自动化,但骗不过你在真屏幕前的眼睛和蓝牙耳机里的耳朵。本协议分两部分:先扫清一个必须拆的雷(静音第一分钟),再走完整体验。

---

## 0. 前置工程闸(交 RELEASE/SOUND 轨,先修再验)

**问题**:Releases 唱片资产未公开发版 → 内网无源 → 实时静音。**但一个新用户 `npx foley` 的第一分钟如果是静的,产品就死在第一分钟。**
**要求:本地唱片自举路径必须成立**——任何网络条件下(含 Releases 未上线、离线、拒绝下载),第一分钟都不许是死寂:
- 无远端源时自动 fallback:内置一枚极小的本地样本唱片(CC0、几百 KB、可进 tarball 的最小可听单元),或至少房间层立即发声并在面板/HUD 提示"唱片架为空,放入 ~/.foley/records/ 或首启下载"。
- **"零配置第一分钟有声"作为发布硬门**入 GATE.md。
- 修毕船长方做下面的完整体验(否则你验的是残缺态)。

## 2. 完整体验走查(戴蓝牙耳机,走一遍新用户的路)

按新用户第一次的顺序走,每步记"是否顺畅＋任何违和/惊喜"。**你是第一个走完全程的人。**

1. **第一分钟**:`npx foley`(或 demo 起点)——起播速度?第一分钟有没有声?看得懂这是什么吗?(妈妈测试:不解释,你能说出它在干嘛吗)
2. **电源仪式**:按电源钮出声——仪式感成立吗?突兀吗?
3. **唱片在放**:真音乐＋机器处置——好听吗?"一首真歌在会老化的机器里放"这句成立吗?音量在蓝牙上舒服吗?
4. **顺风段**:一切正常时,床/针/纸给你的是"岁月静好"还是"无聊"?
5. **风暴段**:张力上升——磁带变旧变闷听得出吗?针压红区、带轴 wow 看得出吗?音画同步吗?
6. **卡碟**:唱针在旋律上跳针——**一耳朵懂吗?**(这是全项目押注最重的一声)
7. **完工**:唱片降速滑停＋针落＋纸停＋静默——"洗碗机时刻"的释然有吗?
8. **等你批准**:琥珀灯呼吸＋音乐让位半格——你会因此回头看屏幕吗?(核心功能验收:它替你值守成立吗)
9. **DUB 仪式**:按键→齿孔→顺孔撕→纸条落台→出 mp4——手感像魔法还是像网页控件?撕的阻尼对吗?
10. **接带成片**:看那支 45 秒 mp4——弧线自明吗?会想发给朋友吗?(传播力自测)
11. **整体**:愿意让它陪你干一下午吗?(G2 那个问题,现在你自己回答)你会向朋友怎么转述它?(这句话就是文案种子)
