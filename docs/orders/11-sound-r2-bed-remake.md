# FOLEY 施工令 · SOUND-R2 床的重做（EAR 战役终裁）

> 阅读对象：Track-SOUND（上下文延续）。本令基于《交接-EAR5-8》与《复盘检讨》两件，全部候裁项在此一次裁清。
> 围栏不变：`sound/`＋声音金测试＋`cli ear`；采样资产新辟 `sound/assets/`（构建期 vendor，运行时零网络不破）。

## 0. 定案与白皮书修正

1. **EAR-10 判决接受**：船长四轮证词的主体 = S1 pad 音色本身。工艺研判佐证：静止裸波形即教科书反例——温暖来自失谐堆叠的拍频、运动的滤波、微随机的"活气"与真实织体；我们造的是两根裸三角波过静止低通。**定性：白皮书规格不足（写了"温暖 pad"没写配方），责任在架构师。**
2. **白皮书修正案（v1.1，随本令生效）**：§2.1 S1 主体改为**织体床**（真采样为体，见 §2）；§7 "v1.0 全合成（已证足）"条款**废止**——"已证足"证的是能出声不是好听；响度纪律出冰箱，配仪器执法（§3 G7）。
3. 链路显影假说降为次要贡献，但主修法对两个假说**同时有效**（织体=掩蔽内容，抬电平=噪声沉底）——修法对残余不确定性稳健。
4. 候船长两句一行（不阻塞开工）：①证伪口——控制组安静版，音**之上**有无颗粒杂音叠加？②设备栏——耳机蓝牙还是有线？

## 2. 床的重做（主修，批次①②合并执行）

**三层结构：**

- **L1 织体体（新主体）**：真采样氛围床——磁带房间声／温和黑胶噪／轻 hiss 织体，2–4 条素材交叉淡化循环，循环点取互质长度防察觉。**资产纪律**：CC0-only；来源限 Freesound（CC0 过滤）、Pixabay、ZapSplat CC0 区、99Sounds 等；**逐条核验授权与出处**（防混入库的盗版声），`sound/assets/LICENSES.md` 登记每个文件的来源 URL／作者／授权；vendored 总量 ≤5MB。**并行保留无资产退路**：滤色粉噪＋脉冲串 crackle 的合成织体作 fallback 与 repo-key 变奏源。
- **L2 和声垫（重做，降为辅助）**：**禁裸振荡器铁律**——任何有音高声部必须过三关方可上总线：①≥3 声部失谐堆叠（±3–10 音分，拍频增厚）；②运动低通（≥2 个互质慢速自由 LFO 推截止／形状）；③轻饱和。电平永远低于 L1。
- **L3 保真总线**：不变，但 hiss 移出 S4 低通（设计自打架修复），磨损维度直达输出。
- **电平与映射**：床总体上抬 +8~12dB，以 G7 校准到位；T→织体密度／滤波／磨损的映射保持——床换了嗓子，仍讲真话。
