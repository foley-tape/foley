2. **F5 v2 语义**：音色（磨损）追随为主，载带模式补 ±2dB 温和响度呼吸；G4 改测磨损参数×T 相关（参数域、确定性）。
3. **旁路复核制**：每里程碑交付 → 独立新会话按设计案全量复核 → 复核档随交付归档 → 船长转呈。置件与修正在合并前落台，署名如实。

## 1. Track-SOUND · SOUND-R3 载带制

### 1.1 播放与装带
- `sound/deck.js`：解码本地音频（decodeAudioData），无缝循环（尾首交叉淡化 ≥1s）；**装带三径**——①BYO：HUD 文件选择器指向本地文件/文件夹（列表轮播）；②出厂带：`sound/assets/tapes/` 构建期 vendor；③无带 → 织体模式自动接管。
- **出厂带纪律**：只收 **CC0 或 CC-BY**（署名清单入 LICENSES.md＋授权页快照列）；"royalty-free 包"凡含禁再分发条款一律不入仓。首选源：FMA 之 HoliznaCC0（CC0 公有领域 lofi 专辑）、free-stock-music（CC-BY 4.0 lofi）、Chosic 逐曲核验。首批 2–3 支，候船长（选带人）盲选定卷。
- **响度归一**：装带即测积分响度（既有 K 加权量具），归一到 **−22 LUFS ±1** 播放基准；G7 改在此基准上执法。

### 1.2 磨损即张力（保真总线作用于音乐）
| 信号 | 载带模式表达（起手值，入 sound-params） |
|---|---|
| T | 高频搁架 0→−8dB；hiss 混入 −54→−40dB；wow 深度 3→18 音分；响度呼吸 ±2dB（低通滤波运动同 R2 曲线族） |
| weather 档 | 小节近似边界（无 BPM 检测：以 2s 网格代拍）切档，雨=蒙尘一档，风暴=再一档＋偶发 dropout |
| WAITING | 走带微滞：playbackRate → 0.985＋滤波收一档＋ASK 动机照旧直通 |
| DONE | **磁带停机**：playbackRate 1→0 约 1.2s 俯冲入 ≥2s 真静默（正格终止感由俯冲承担） |
| STUCK | **环形缓冲跳针**：捕最近 0.5s 音频循环播放；STUCK_CLEARED（ok 型）解除并轻接带音；expiry 型静默解除 |
| RESOLVE | 磨损瞬时回撤 30%（云开）＋既有和弦音色可选叠加（音量低于音乐） |

### 1.3 前景切换
- 载带时前景 = **无音高 foley**（翻页/铅笔/打字机/卡座咔哒）＋呼唤三音（跳针/ASK 动机/终止静默）；五声拨弦仅织体模式保留（避免与任意调性的音乐打架）。foley 不做节拍量化——房间声不跟收音机对拍。

### 1.4 机器耳朵 v3
- G1–G3 照旧 active（stop 静默／trim 有效／基准响度改测归一后音乐床）。
- **G4v2（active）**：五带渲染，磨损参数向量（搁架 dB、hiss dB、wow 深）与 T 的相关 r ≥ 0.8（参数域，确定性）。
- **G8（active，新）**：STUCK 段渲染呈周期性自相关峰（跳针在响的机器证明）；CLEARED 后自相关峰消失。
- G6 织体占用度仅织体模式适用。`cli ear` 分模式出表。
