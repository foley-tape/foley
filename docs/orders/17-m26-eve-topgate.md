# FOLEY 施工令 · M2.6 发布前夜(四轨 · 首航闸门最高优先)

> 本令在 GATE.md 上**新增一道最高优先闸门**,置于所有其他闸门之前:
> **GATE-0 · 首航与保真**:船长走完完整流程且视觉保真达标之前,禁止拆闸发布。

## §0 Track-CAPTAIN · 首航(最高优先,唯船长可做,喂养其余三轨)

1. **先解锁真声音**:临时将 Releases 唱片资产手动落入 `~/.foley/records/`(或发行轨提供本地 unlock 脚本),使本地实时有源——**发布前必须听全完整声音**。
2. **走完整流程并录屏**,逐站记不适:
   - `npx foley` 冷启动 → 开机电源钮仪式 → 听全床/唱片/前景三层
   - 跑一场真 storm → 亲历针/墨线/带轴/琥珀/跳针/DONE 滑停全套
   - 按 DUB → 齿孔提议 → 撕 → 导出 mp4 → 看成片
   - 拒绝下载唱片一次 → 验房间层降级路径
   - 藏标签页/久置 → 验深睡与唤醒
3. 产出 **首航日志**(`docs/records/maiden-voyage.md`):逐站"符合预期/不适/疑似 bug",附录屏。此日志是本轮最高价值产出——**它定义了发布前还必须修什么**。
4. 首航中若发现阻断级体验缺陷 → 直接升 P0,发布顺延,不带病出港。

## §B Track-STAGE · 质感大修(拟物保真,发布门)

船长所列八处塌陷,逐条修缮(验收铁律:**微距截图放大到 4K 满屏,须像一张真机照片**):

| # | 缺陷 | 修缮方向 |
|---|---|---|
| 1 | 整体发虚 | 渲染锐度审查:半分辨率 shader 与器件清晰度的平衡复核;器件边缘不该被镜头层糊掉 |
| 2 | 阴影塌陷 | 接触阴影(器件落在面板上的贴地暗)与投射阴影分层;暗部不能死黑成片 |
| 3 | 坐标纸违和 | 纸张光学:纸面微反光/纤维质感/油墨吃进纸的洇散;网格线印在纸上而非浮在表面 |
| 4 | 纸缺光学反应 | 纸随画外暖光有极淡的明暗过渡,不是一张死平的贴图 |
| 5 | VU 无玻璃罩反射 | 烟色玻璃面罩:斜向高光带 + 边缘暗角 + 针在玻璃下的景深三明治 |
| 6 | 黑针裸露 | 针封进表壳:玻璃罩之下、表盘之上的封闭层次,物理仪器的密闭感 |
| 7 | ASK 开关扁平 | 拨动开关厚度:侧壁高光/顶面反光/拨柄投影,电木的体积感 |
| 8a | 指示灯刻字松散 | DONE/MAIN 刻字排版收紧:字距/基线/字号按 Univers 字法重排 |
| 8b | DUB 键像塑料 | 电木/金属按键质感:沉甸甸的阻尼、边缘倒角高光、按下的机械行程——不是注塑件 |
