## 0. 复盘裁决（全部生效）

1. **脊柱入宪**：琥珀宪法升 **v1.3**，新增画侧总纲三句——**"引擎管物理，舞台管光学；凡在舞台上私造物理，都是对磁带说谎。钟在包里。渲染只做恒迟一包的重建。"** 置于宪法序言位。
2. **交互法（逐字入宪）**：一切面板交互必须是这台机器的主人会对它做的动作（凑近、撕纸、轻敲表头玻璃）；界面味交互（按钮/菜单/拖拽滑块）不得上面板，dev 抽屉除外。
3. **体温法（入宪）**：这台机器不许让风扇说话——功耗是音画设计的一部分。常驻 CPU 目标 < 5%，进 M-S4 验收。
4. **灯之归属（界线亲画）**：器件承载**读数**（值＋方差，器件法管辖）；布景承载**氛围**（光法与灯位法管辖）。氛围可以骑在光上，读数永远不行。画外灯是布景——weather 上灯合法。
5. 发布颗粒：屏内坚持一次成像；**导出按介质适配**——GIF 静态颗粒单帧化、高码率视频保动态颗粒。照片本来就不是机器本身，按介质适配是诚实不是妥协。细则候预告片轮。
6. 多会话架构：架构师倾向**多 spawn 多机架**（一会话一台机器，协议不动）；预告片轮携体温实测数据终裁。
7. 探讨一（DONE 落针）已裁归引擎侧，Track-SOUND 顺手单执行；探讨二 fixtures 重放**授权**（封版 CLI 只读重放五带，产物落 `stage/fixtures/`，删除 WAITING⇔pendingAsk 推导）。

## 1. M-S4 施工范围（按值钱程度排序）

1. **今晨的纸**：开机先以回放管线吃当日 `cli live --out` 产物铺纸（8× 铺纸与实走全等的既有红利），追平后无缝切 SSE。零引擎改动、零协议改动，全在围栏内。一台以"纸带即时间轴"立论的机器，不再开机失忆。
2. **体温预算**：镜头 shader 降 30fps；走纸增量绘制（墨只追加：平移＋补新墨＋笔桥，脏矩形）；`document.hidden` 渲染整停（状态照走）。交付含实测：常驻 CPU、能耗档，目标绿档 <5%。
3. **weather 灯档**：画外灯随天气分档只压暗（雨降一档温度收一成亮度、风暴再降），小节边界切换与声侧同拍。光法之内自由。
4. **性格照管线转正**：`stage/portraits.json` 存五带窗口定义（带、prefill、起点、秒数），一条命令全量重拍——素材诚实条款机械化，架构师验帧获得固定机位。
5. **IDLE 深睡台阶**：IDLE 持续 5min 再降一档、呼吸性暗角放慢、只余待机粒；唤醒走钨丝速度。
6. fixtures 重放清账（§0.7 授权项，十分钟）。

## 2. 品味验收的影子指标（复盘 §1⑥ 推广）

每句品味验收配一个机器可判影子，本轮先立三条（informational 首轮）：
- 针尖峰速上限（渲染端实测 ≤ 引擎弹簧理论峰速 ×1.05——光学不许比物理快）；
- 恒迟实测 ≤ 60ms（live 与回放两测）；
- 体温绿档（§1.2 的 CPU 判据）。
