## 0. 事故上下文（60 秒读完）

产品是一台把 coding agent 会话实时演奏成 lo-fi 磁带声画的仪器。声音架构见 `docs/canon/`（或根目录）的《感官设计白皮书》：连续床（S1 基底/S2 律动/S3 张力/S4 保真总线）＋离散前景（拨弦/和弦/跳针）。
前一施工方在声音相连续四轮被船长实听否决（"床是大噪声"），三轮自称修好全部与耳朵矛盾。升级审计已定根因（下节），**代码按令未再动**。你接手的第一原则：**不信任何"已验证"的历史宣称，只信渲染出来的音频。**

## 1. 根因（已实证，勿重查，直接修）

`cli/probe.ts` 生成页 S1 段，自第一版起：

```js
bg1.gain.value=0.15; bg2.gain.value=0.1;      // LFO 绝对深度
b1.connect(bg1); bg1.connect(G.s1.gain);       // 直连增益参数
b2.connect(bg2); bg2.connect(G.s1.gain);
```

S1 设计电平仅 ~0.035，呼吸副肺 ±0.25 —— **比正身响 17dB**。它同时解释四轮症状：床=大噪声、trimDb 无效（只缩放 0.035 项）、停止后仍响（停止归零的是自动化值，管不着外接信号）、音乐退场噪声留下。
前任三轮失明的机理：**`AudioParam.value` 读数不含外接信号贡献**（Web Audio 规范）——账本全零，节点仍在出声。

## 2. 修法裁决：方案 B（结构免疫）

- 呼吸 LFO **不得直连任何增益参数**。改为调制独立乘法级（gain×gain 串联）：`body → breathStage → out`，LFO 摆动 breathStage 于 `1 ± breathDepth`。呼吸在数学上永远是正身的百分比，不可能反相、不可能压过正身。
- `bed.breathDepth` 入 sound-params.json（相对值，起手 0.12，可调域 0.05–0.20）。
- **音频图注册表（重构执照，船长已授权算力）**：把声音从 probe 生成页中析出为独立模块 `sound/`（probe 变薄壳，未来舞台同源复用）。规矩三条：①一切 Gain/Filter/Source 节点建于注册表，拓扑在模块头部用注释画全图；②`stop()` 与 `trim` **遍历注册表**作用于每个可发声支路——僵尸节点从结构上灭绝；③外接调制（LFO 等）一律走乘法级，直连参数为禁手。
- 允许一次时间盒调研（≤半天）：读 Tone.js 源码中 context/param/dispose 管理的成熟做法作参照，**借骨不搬库**（运行时零依赖照旧）。

## 3. 机器耳朵（本令核心交付：`cli ear` ＋ 声音金测试）

用 `OfflineAudioContext` 离线渲染，无人耳判定。**任何"修好了"的宣称必须先过机器耳朵。**

| 门 | 判据 | 状态 |
|---|---|---|
| G1 停止即静默 | 播放→停止，其后 1s 渲染 RMS < −60 dBFS | **active** |
| G2 总闸有效 | trimDb ±12dB 两次渲染，床段 RMS 差 ≥ 10dB | **active** |
| G3 床响度守设计 | 五带各 30s，床 RMS 落 sound-params 设计值 ±3dB | **active** |
| G4 床-张力相关 | storm 带渲染，床 1s-RMS 包络 vs T 曲线 Pearson r ≥ 0.6 | informational 首轮 |
| G5 呼唤穿透 | 跳针触发时，其频谱专区能量高于床 ≥ 6dB | informational 首轮 |

- G1–G3 直接编码本次事故的三个症状，立即拦路；G4/G5 按判据试用期法首轮记分不拦。
- 全部入 golden 与 `cli ear`（PASS 表输出，携 sound-params hash）。渲染必须确定性（OfflineAudioContext 定长渲染天然满足）。

## 4. 船长听感协议（权益保护法）

1. **船长的耳朵是验收权威，不是调试仪器。**机器耳朵全绿之前，不得申请实听。
2. 每次实听申请必须附：`cli ear` 全绿输出（含哈希）＋ 一句"这次你该听到什么不同"。
3. 一批修复只许请一次耳朵。听感台账沿用 EAR-n 编号追加。

## 5. 顺手单（引擎唯一例外，机器耳朵全绿后执行）

**DONE 断电落针**：`phase=DONE` 时引擎将 needle 弹簧目标置 0（走既有慢放翼），**S 账本照常衰减不动**。理由：马达断电、表头失励磁——归零是仪器的诚实物理，不是数据造假；"洗碗机时刻"由此干净。加一条金测试；五带重跑确认各 band 无触动（band 全是 T 基，理应无扰）；FEEDBACK 记录。
