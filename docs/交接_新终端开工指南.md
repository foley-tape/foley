## 三、伤疤（会咬你的七件事）
1. **CSS 双表战争**：stage.css 旧 ID 规则压过 plate.css 类规则——新层定位怪异先 getBoundingClientRect 对坐标再 grep 两表。三案判例是你整固批的施工图。
2. **验收≠产品**：只验"会动"不验"完整故事"已翻车七次。断言取窗要看页面语境（demo=带妆待命：storm 定妆包让待机针位非零/魔眼先热）。
3. **朴素像素检测器不可信**：盲窗/阈值/降采样三次翻车——DOM 反解定位＋全分辨率裁片＋目验。
4. **强制值验收竞态**：setProperty 会被活包络写者覆写——用免手势页或钳住写者；验 LINE 熄必须 `live.es.close()` 真断（活 SSE 50ms 一包顶回 live）。
5. **验声工艺**：`--autoplay-policy=no-user-gesture-required`；手势→选带 ≥2.5s；换码必清 profile（serve 已 no-cache 仍防万一）。
6. **体温法红线**：值不变零写、停即零绘制；任何新动画层跑 `?perf=1` 60s，稳态 LONG 必须 0。
7. **一次一项＋证据回写**：完工＝船长机证据入 ~/Desktop/至架构师/＋账本证据栏一条；发现不修不展开，入发现栏。
