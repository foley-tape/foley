# 器件户口册
立法（BATCH3 乙-1·2026-07-13 首册）：**每一件可见器件在此登记——名／职能／数据源／待机行为／交互反应。无户口者下次渲染逐出面板。**
本册即船长十分钟的答案清单（丙验收线："运行态的机器像活物，且我能说出每件东西是干嘛的"）。

## 四、边界属性（不是器件，是律）
- **REC 全脸律（⑥e）**：信箱重构图时，全机唯一的红必须全脸入画。现状在案：rec_jewel 心 y＝−0.28%＝**当前削头**（约半脸出画顶缘）——候重构图时一次纠正，不单独动信箱。
- **翻字牌 12 格硬截断＋1150ms 落定帽**（值班律·金测㊿㊾）——超长曲名截、全牌帽内到站。
- **修宪甲**：全机唯一信号亮红归 REC；照度 ≤0.15 红宝石属基底照度不构成信号红（POST 灯检抬 0.26/0.34 属灯检非信号）。
- **辊限步 0.14 rad/帧**（值班律：防风暴频闪轮）；**VU 定标 0VU≡−20dBFS≡画上红区界**（三点定标 −20/−30/−17）。
- **体温法**：一切器件值不变零写、停即零绘制——待机的机器该是静的。
