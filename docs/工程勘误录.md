| # | 勘误 | 细节与处方 | 案发 |
|---|---|---|---|
| 1 | headless 虚拟时两坑 | `--virtual-time-budget` 下 **rAF 不走**（动效须 ?autoplay 自泵）；**2D canvas 不再向合成器上屏**（截图恒首帧，墨账用 `getImageData` 验真）。真浏览器无此二坑 | decree13 墨线"消失"一小时假案 |
| 2 | node 内置 WebSocket（undici） | 不持事件循环（无定时器的 await 空转即退出 exit 13——挂心跳 interval）；单帧 >~1MB 或累计 ~7MB 即 1006 断连。CDP 采集一律 npm `ws`（maxPayload 调大）＋大回包分片 evaluate（≤200KB） | decree14 闸材料采集 |
| 3 | macOS 遮挡停合成 | 窗口被遮挡 → Chrome 停止合成 → screencast 饿死、CPU 采样假 0%。处方：`--disable-backgrounding-occluded-windows --disable-renderer-backgrounding` ＋ `Page.bringToFront`，采样前先验 visibilityState 与帧计数在走 | P0-1 假清白 G 轮 |
| 4 | h264 偶数律 | `--app` 窗高被标题栏啃成奇数 → libx264 拒收。兜底 `crop=floor(iw/2)*2:floor(ih/2)*2` | 闸材料 1680×1019 |
| 5 | 金属只吃 glossy | Blender metallic=1 材质对 `visible_glossy=False` 的灯完全免疫——34W/300W 全黑同因。牌灯类必须开 glossy 可见 | 走带牌 np_lamp |
| 6 | TCC/macl | ~/Downloads 无 `com.apple.macl` xattr 的文件对终端进程整体 EPERM（读/改名皆拒）。**工作文件入 repo**（账本入仓的直接动因） | FOLEY_LEDGER.md 首发被咬 |
| 7 | 相机锁死＝坐标即常量 | 场景板相机锁死后 `world_to_camera_view` 归一化坐标可当常量对位 CSS——"动态层靠算不靠瞄"的根基（坐标单源 §8 将其自动化） | decree13 plate.coords.json |
| 8 | 混合层×动画 | mix-blend 元素一动（或其下层动），整幅每帧重混合。动画元素要么撤 blend＋will-change 上合成器，要么与被混内容同缓存子树 | P0-1 呼吸辉光 |

| 10 | 水位线吞史 | "connect 即喂最新态"与"按 t 去重回灌历史"同存时，最新态把水位钉在现在→历史整批被吞。回灌前必须回拨钟基与水位 | P0-3 病根② |
| 11 | Python heredoc 中文 | `python3 - <<EOF` 无编码声明遇中文即 SyntaxError——首行加 `# -*- coding: utf-8 -*-` | 账本写回轮 |
| 12 | git quotePath 转义 CJK | `git ls-files`/`status` 默认 `core.quotePath=true` → 中文路径八进制转义（`\346\…`）。凡脚本按名匹配中文路径须 `ls-files -z`（NUL 分隔不转义）或 `-c core.quotePath=false` | 席三 check-ledger-writeback.mjs |
| 13 | 浏览器闸脏树 | 浏览器/CDP 回归闸入默认 `npm test` 会往工作树写截图/日志（脏树）。守卫环境变量 `FOLEY_GATE`：置位才落工件，例行只跑断言不落图 | 席三 latecomer 入 npm test |
| 14 | 预算单位 bytes≠MiB | 包体/尺寸预算单位显式 **bytes 十进制**：2,000,000 B ≠ 2 MiB(2,097,152)。断言写死进制与单位，勿用口语"2MB"（对审曾因此歧义重签） | 席三 check-pack-budget 对审 |
