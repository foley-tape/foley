# 诊断口注册表
律（LEDGER·增补包 v2 §5）：**新口不登记不算存在**。每口必填：参数、打开什么、给谁用、"保证不进正常路径"的实现位置。
诊断口＝只为验收/排障存在的开关；正常用户永远不带参开页。
